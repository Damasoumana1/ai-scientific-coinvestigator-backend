# 🚀 AI Scientific Co-Investigator | L'avenir de la recherche assistée par l'IA

## 🌟 Introduction
Le **AI Scientific Co-Investigator** est une plateforme revolutionnaire conçue pour transformer la manière dont les chercheurs interagissent avec la littérature scientifique. En combinant le raisonnement avancé de l'IA avec une architecture RAG (Retrieval-Augmented Generation) robuste, le système devient un véritable partenaire de recherche.

## 🛠️ Fonctionnalités Clés

### 🔍 Explorez & Gérez vos Projets
- **Gestion Stucturée** : Organisez vos recherches par projets dédiés.
*   **Téléversement Intelligent** : Support complet pour les PDF avec extraction textuelle de haute précision.
*   **Discovery (ArXiv Integration)** : Ne perdez plus de temps à chercher sur le web. Recherchez, téléchargez et analysez des articles scientifiques directement depuis ArXiv via l'interface.

### 🧠 Analyse Comparative & Détection de Contradictions
- **Analyse Multi-Documents** : Ne lisez pas les papiers un par un, analysez-les tous ensemble !
*   **Détection Automatisée de Contradictions** : L'IA identifie les incohérences entre différentes études pour souligner les points de friction dans la science actuelle.
*   **Identification des "Gaps"** : Le système trouve ce qui manque dans la littérature pour suggérer de nouvelles pistes de recherche.

### 🧪 Design Expérimental Assisté (Moteur K2 Think)
*   **Génération de Protocoles** : Créez automatiquement des protocoles expérimentaux détaillés basés sur vos hypothèses.
*   **Auto-Critique & Optimisation** : Utilisation du moteur **K2 Think** pour générer plusieurs versions, les critiquer et ne retenir que la plus robuste.
*   **Variables & Risques** : Analyse automatique des variables indépendantes/dépendantes et évaluation des risques.

### 📊 Visualisation & Traçabilité

## 🏗️ Stack Technique (State-of-the-Art)
- **Frontend** : Next.js 13, Tailwind CSS, React Flow (Déployé sur **Vercel**).
*   **Backend** : FastAPI (Python), LangGraph pour l'orchestration des workflows (Déployé sur **Hugging Face Spaces**).
*   **Bases de données** : 
    - **Supabase** (PostgreSQL) pour les données structurées.
    - **Qdrant Cloud** pour la recherche vectorielle (RAG).
    - **Upstash Redis** pour la gestion des tâches asynchrones.
*   **IA** : K2 Think API pour le raisonnement profond + OpenAI pour les embeddings.
